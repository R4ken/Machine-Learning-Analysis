# Machine Learning analysis

Analysis of regression and classification tasks with preprocessing and automatic hyperparameters selection

## Setup
Project is written in jupyter notebook files
To setup kernel with needed libraries enter following commands (uv recommended)
```bash
uv sync
uv run python -m ipykernel install --user --name=ml-kernel --display-name "Python (ML)"
```
Then open files in your choosen IDE, for example
```bash
jupyter-lab .
```
